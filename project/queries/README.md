# Project: Queries

This is a team project. Work on it with your assigned team. Only one of you
needs to turn it in — but make sure someone does!

## Objective

* Develop a list of queries that you would like to run against your database.

## Details

Develop a list of at least 10 queries that one would like to be able to run
against your database. This list should include:

* Simple queries against one of the entities.
* At least one query that involves your junction table.
* A few update queries to add entries in your database.

## What to hand in

Upload a text file with your list of queries. Do *not* implement these queries
in SQL yet (this is going to be the topic of the next project assignment). The
file should consist only of descriptions of queries in English prose.
