# Project: PHP

This is a team project. Work on it with your assigned team. Only one of you
needs to turn it in — but make sure someone does!

## Objective

* Allow a user to interact with your database through a web frontend

## Details

Support at least two select queries and one update query. The entry point into
the `tree` frontend should be `https://localhost/tree/`.

Present a menu of choices to the user in `tree/index.html` and pass parameters
using the `<form>` tag.

From the answer pages, provide the user the ability to return to the main menu
(`tree/index.html`).

## Hint

For testing, place your HTML and PHP files in `/Applications/XAMPP/htdocs/tree`
of your virtual machine.

## Extra credit

You can earn extra credit, if you use Cascading Style Sheets (CSS) to improve
the presentation and style of your solution.

## What to turn in

*When ready, show your work to the instructor.*

But also, upload a zip file with all the HTML/PHP files you have created.

The beginning of every file should include the names of all participants in your
group (if one person didn't contribute, don't include their name).
